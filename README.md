Chrome / Edge 扩展，使用 DeepSeek AI 自动分类整理 B 站收藏夹视频。

## 功能

- 一键整理：点击扩展图标，配置后直接启动
- 两种模式：移动模式（视频只归一个收藏夹）、复制模式（视频可归属多个收藏夹）
- 分批处理：大量视频自动分批，避免 AI 输出截断
- 智能重试：网络或 API 异常自动重试，可配置次数
- 自定义要求：支持输入特殊分类规则，AI 优先遵循
- 优先匹配：AI 优先将视频归入已有收藏夹，减少重复建夹
- API 可配：支持自定义 API 地址和模型，兼容第三方中转

## 安装

1. 下载 `extension` 目录
2. 浏览器打开 `chrome://extensions/`（Edge 为 `edge://extensions/`）
3. 开启「开发者模式」
4. 点击「加载已解压的扩展程序」，选择 `extension` 文件夹

## 使用

1. 点击浏览器工具栏中的扩展图标
2. 在弹窗中填入 DeepSeek API Key 并保存
3. 打开 B 站个人空间，进入某个收藏夹页面
4. 选择整理模式，填写自定义要求（可选）
5. 点击「开始深度整理」
6. 页面右上角出现状态面板，实时显示进度

API Key 获取：https://platform.deepseek.com/ → 注册登录 → API Keys

## 整理模式

| 模式 | 行为 | 适用场景 |
|------|------|----------|
| 移动模式 | 视频从当前收藏夹移入目标分类 | 每个视频只归一个收藏夹 |
| 复制模式 | 视频复制到匹配的分类收藏夹，仍保留在原收藏夹 | 交叉分类，一个视频归属多个收藏夹 |

复制模式下 AI 会将视频放入所有沾边的分类。如分类过于分散，可在自定义要求中约束（如"每个视频最多归属2个分类"）。

## 费用参考

使用 DeepSeek Flash 模型，整理约 3,400 个视频（30+ 收藏夹），消耗约 1,215,240 token，费用约 1.18 元。

## 常见问题

**网络不稳定**：在弹窗「高级设置」中调大「最大重试次数」。

**AI 分类不准**：对分类不准的收藏夹再跑一次即可，已匹配的视频不会被移走。也可在自定义要求中补充规则。

**扩展无法连接页面**：刷新 B 站收藏夹页面后重试。

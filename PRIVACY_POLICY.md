# B站AI收藏夹整理 - 隐私策略

## 数据收集说明

本扩展仅收集以下数据，且所有数据均存储在用户本地浏览器中：

1. **API Key**：用于调用 AI 服务进行视频分类。该密钥仅存储在浏览器本地 storage 中，不会上传到任何服务器。

2. **扩展配置**：包括 API 地址、模型名称、批次大小、重试次数等用户自定义设置。

3. **B站视频信息**：扩展会临时读取用户收藏夹中视频的标题和简介，用于 AI 分类分析。这些数据仅在整理过程中使用，不会被存储或传输到第三方服务器。

## 数据使用说明

- 所有用户数据仅用于实现扩展的核心功能（AI 智能分类整理）
- 不会收集、存储或传输任何个人身份信息
- 不会收集用户的浏览历史、位置信息或其他敏感数据
- API Key 仅用于调用 API，不会被用于其他用途

## 第三方服务

本扩展使用 API 进行 AI 分类。调用 API 时，视频标题和简介会被发送到 LLM 服务器。请参阅 各LLM厂商 的隐私政策了解其数据处理方式。

## 数据存储

所有配置数据存储在 Chrome/Edge 浏览器的本地 storage 中，用户可以随时清除这些数据。

## 联系方式

如有隐私相关问题，请通过 GitHub Issues 联系开发者。
